### Prerequisites
Ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v14 or higher)

### Installation

1. Clone the repository:
   
   ```bash
   git clone https://github.com/bbusn/gassion.git
   cd gassion
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```
